# Student Group Management System

A simple Python project demonstrating object-oriented programming concepts:
inheritance, encapsulation, custom exceptions, hashing, and modular structure.

## Features

- Human → Student inheritance
- Group class for managing students
- Search and delete students by last name
- Group size limit with custom exception
- Correct comparison and hashing for Student objects
- Clean separation into modules

## Project Structure

project/
│
├── human.py
├── student.py
├── group.py
├── exceptions.py
└── main.py


## Technologies

- Python 3
- OOP principles
- Custom exceptions
- `__str__`, `__eq__`, `__hash__` magic methods
- Sets for unique object storage
